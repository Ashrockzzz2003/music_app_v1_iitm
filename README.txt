Follow the instructions to run the app, make sure you have python 3.7 or above installed on your system.

--------------------------------------------------------------------------------------------------

Step 1 (MacOS, Linux Users)

python -m .venv venv
source .venv/bin/activate
pip install -r requirements.txt

Step 2

Step 3
